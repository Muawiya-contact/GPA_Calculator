# GPA Calculator

A simple, clean GPA and CGPA calculator for university students. No sign-up, no server, just open and use.

---

## What it does

- Calculates your semester GPA from marks and credit hours
- Tracks cumulative CGPA across semesters
- Lets you edit the grading scheme to match your university
- Saves semester history in your browser
- Works on phone and laptop, day or night (dark mode included)

# How to use

1. Download all project files:
   - `index.html`
   - `style.css`
   - `script.js`

2. Make sure all files are in the same folder

3. Open `index.html` in any browser

4. Add your subjects, enter marks and credit hours

5. Click:
   - **Calculate Semester GPA** → to see current GPA
   - **Calculate CGPA** → to see overall CGPA

6. Enter previous CGPA if you want CGPA calculation

7. Refresh page to reset or change data

## Grading scheme

Default is a standard 4.0 scale. You can change it from the sidebar under **Grading scheme** — edit the cutoffs, grade letters, and points to match whatever your university uses.

## Notes

- Everything stays in your browser. No data goes anywhere.
- Works fully offline after the first open.
- Built for NFC-IET and VU students but works for any university.

---

Made with plain HTML, CSS, and JavaScript. No frameworks, no dependencies.
